Retiro Gelish
$100
El servicio de Retiro de Gelish consiste en la eliminación profesional del esmalte de gel aplicado en las uñas. Este procedimiento se realiza de manera cuidadosa para evitar dañar la superficie de las uñas naturales.

Generalmente, se utiliza una solución específica para disolver el gel, que se aplica con un algodón y se cubre con papel de aluminio para acelerar el proceso. Después de un tiempo de espera, se retira el gel suavemente con una espátula o un palito de naranjo.
